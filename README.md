<h1>⏳ Pomodoctivity</h1>

<p>
<strong>Pomodoctivity</strong> is an advanced productivity web application built using 
<strong>HTML</strong>, <strong>CSS</strong>, and <strong>JavaScript</strong>.  
It combines the Pomodoro technique with performance tracking and weekly analytics to help users monitor and improve their focus.
</p>
<hr>

<h2>🚀 Key Features</h2>
<ul>
    <li>Customizable focus and break durations</li>
    <li>Real-time countdown timer</li>
    <li>Automatic session tracking</li>
    <li>Daily statistics (total hours focused & sessions completed)</li>
    <li>Weekly performance analytics</li>
    <li>Interactive charts displaying weekly data</li>
    <li>Persistent data storage using Local Storage</li>
</ul>
<hr>

<h2>📊 Productivity Dashboard</h2>
<ul>
    <li>Total hours focused today</li>
    <li>Total completed sessions today</li>
    <li>Weekly focus hours visualization</li>
    <li>Weekly session completion chart</li>
</ul>
